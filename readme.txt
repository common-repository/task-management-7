== Description ==

Connects WordPress with your Restyaboard instance through Contact Form7 plugin. Restyaboard is an open source Trello alternative to manage todos, tasks, supports, etc. Using the integration, you can feed user submissions to Restyaboard to manage them on a visual kanban board. Capture customer feedback, inquiries, order requests, etc from your WordPress, but manage them through Restyaboard to bring workflow and process management to your business.


If you have any problems, questions, or issues about this plugin then please create a support request and we will get back to you quickly.

Note: You'll need a cloud or self-hosted Restyaboard instance to use this plugin.


== Installation ==

= Automatic Installation =
> 1. Sign in to your WordPress site as an administrator.
> 2. In the main menu go to Plugins -> Add New.
> 3. Search for Plugin Name: Task Management 7 - Contact Form 7 plugin for Restyaboard, the open source Trello alternative, and click install.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin is a sub-plugin of the famous Contact 7 plugin. It helps you to push your tasks/inquiries/etc from your website form to your Restyaboard instance.

= What is a Restyaboard? =

Restyaboard is an open source alternative to Trello. You can self-host it on your machine, intranet, or your cloud. A 1-click cloud install option is available too.
